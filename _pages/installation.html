Installation

You can install puffin.js in two ways:

With NPM

{% highlight terminal %} npm install puffin.js --save {% endhighlight %}

With Git

{% highlight terminal %} git clone https://github.com/Treast/puffin.js.git mv ./puffin.js/dist/puffin.min.js ./my-project {% endhighlight %}
